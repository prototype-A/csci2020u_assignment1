# Spam Checker
CSCI 2020U Assignment 1


## How to Compile

### Requisites
Git installed/Repo downloaded as zip and extracted
Gradle installed

### Commands
In the cloned/downloaded repo's directory:

Compile the program:
`gradle jar`
Navigate to the executable jar's directory:
`cd build/libs`
Run the program:
`java -jar csci2020u_assignment1.jar`
